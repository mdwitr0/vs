## Отслеживание лицензий

### 3.1 Webhook-получатель от poiskkino.dev

**Функционал:**
- Endpoint для приема webhook от твоего сервиса
- Валидация входящих данных
- Обработка изменений в полях лицензий
- Обновление базы отслеживаемого контента
- Логирование всех изменений

**Структура webhook payload:**
```json
{
  "movie_id": "string",
  "field_name": "string",
  "new_value": "any",
  "timestamp": "timestamp"
}
```

**Оценка:** 8-12 часов

### 3.2 Хранилище данных о лицензиях

**Структура:**
```json
{
  "kp_id": "string",
  "title": "string",
  "year": "int",
  "licenses": "array",
  "last_updated": "timestamp",
  "external_ids": "object"
}
```

**Оценка:** 4 часа (интеграция в существующую базу контента)

### 3.3 Интеграция с системой сканирования

**Логика:**
- При получении обновления лицензий проверяем наличие этого контента в индексе
- Если нет - добавляем

**Оценка:** 6-8 часов

### 3.4 Фронтенд: отображение статуса лицензий

**Функционал:**
- Колонка в таблице контента с информацией о лицензиях
- Фильтр по наличию лицензий
- Индикатор давности последнего обновления
- Детальная информация о лицензиях на странице контента

**Оценка:** 5 часов
